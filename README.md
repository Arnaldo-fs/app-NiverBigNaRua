# 🎉 App de Aniversários

Um aplicativo simples feito em Flutter para visualizar aniversários cadastrados. <br><br>

## Sobre o projeto
Este app permite visualizar uma lista de aniversariantes do mês carregados diretamente de uma planilha do Google Sheets, além de receber uma notificação do aniversariante do dia em questão, caso o usuário se esqueça <br><br>

## Tecnologias utilizadas
- Flutter
- Dart
- Google Sheets (como base de dados) <br><br>

## Base de dados
- Os dados são consumidos via Google Sheets no formato CSV (Sheet criado exclusivamente para teste do App):

https://docs.google.com/spreadsheets/d/1Lsjlpr5UWE3pvcMLXxLYlW3Dw1Gx_HbhozafsbhQQcU/export?format=csv  <br><br>

## Funcionalidades
- Listagem de aniversariantes
- Integração com Google Sheets
- Interface simples e funcional <br><br>
